\section{\LaTeX-style PreTeXt}\label{tryit-latex}

This section is written in \LaTeX-style PreTeXt markup. You can edit
this content using supported LaTeX syntax.

\[
    \left|\sum_{i=0}^n a_i\right|\leq\sum_{i=0}^n |a_i|
\]

When you preview, the source is converted to PreTeXt XML and rendered
as accessible HTML.
