title: Markdown-style PreTeXt
---

This section is written in Markdown-style PreTeXt markup.
You can edit this content using supported Markdown syntax. For
authors not familiar with LaTeX or PreTeXt, or coming from other
Markdown-inspired languages like Quarto, this is a great way
to get started.

Of course, you still can use LaTeX syntax for authoring
mathematical expressions:

$$
    \left|\sum_{i=0}^n a_i\right|\leq\sum_{i=0}^n |a_i|
$$

When you preview, the source is converted to PreTeXt XML
and rendered as accessible HTML.

Theorem:
    PreTeXt can be used in Markdown-style mode.
    Proof:
        See this section!
